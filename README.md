# chess-bot

## Install

```
poetry install --no-dev
```

## Developer Settings

```
poetry install
pre-commit install
```

## How to Execute
```
git clone https://github.com/ainize-team/chess-bot.git
GUILD_ID={Discord Server ID} TOKEN={Discord Bot Token} python src/bot.py
```

## Slash Commands
```
/start {Black, White}
/move {Legal Move}
```
